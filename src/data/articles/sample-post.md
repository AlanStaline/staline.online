# 开始使用 Astro 构建现代网站

Astro 是一个现代化的静态网站构建工具，专注于性能和开发体验。

## 为什么选择 Astro？

- **零 JS 默认**：默认情况下不发送 JavaScript 到客户端
- **组件 islands**：选择性交互，只加载需要的 JavaScript
- **多框架支持**：支持 React、Vue、Svelte 等组件

## 快速开始

```bash
npm create astro@latest
```

享受构建网站的乐趣吧！
